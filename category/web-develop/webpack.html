Webpack คืออะไร?

Posted on October 1, 2023

ถ้าเคยสร้าง modern JavaScript application ต้องเคยได้ยินคำว่า “webpack”

Webpack เป็น static module bundler สำหรับ JavaScript applications โดยจะใช้โค้ดทั้งหมดจาก application และทำให้สามารถใช้งานได้ใน web browser

Modules คือ ชิ้นส่วนโค้ด (chunks) ที่สามารถนำมาใช้ซ้ำได้ ซึ่งสร้างขึ้นจาก JavaScript, node_modules, รูปภาพ และสไตล์ CSS ของแอป ซึ่งจัดทำเป็น package เพื่อให้ง่ายต่อการใช้งานบนเว็บไซต์

Webpack จะแยกโค้ดตามลักษณะการใช้งานในแอป และด้วยการแยกย่อยความรับผิดชอบแบบ modular ทำให้การ manage, debug, verify และทดสอบโค้ดง่ายขึ้นมาก
